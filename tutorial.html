Tutorial



1. Creating Room

Enter username and room name and press start.
If you leave them blank, it will be created randomly.

You can add players with buttons.

When you add a player, a computer will be added by default.
you can change it to host another guest.

You can add players up to 4, and you can have 2 guests total.

You can change computer`s character by clicking the character icon

hosting another player

A guest successfully added


2. Joining Room

If you choose 'Join' in game lobby, it will show list of rooms waiting for other guest.

After joining the room, you can choose your character.

Be sure to press ready if you are ready so the host can start the game.

※ If one player quits the game, the game automatically stops.


3.Game Interface

Message box will be shown if you click write message button.

You can click the status effect icon to see what it is.

For more information about status effects, click here.

Using dice control

dice control lets you control your dice. you can use once in every 5 turns.

if you click dice control button and choose a number instead of throwing dice, the dice will be controlled by your choice.

4.Gameplay

The map

When you arrive on a square after throwing dice, you will get the effect of an obstacle on that square.
There are beneficial obstacles, harmful obstacles, and strategic obstacles.

To see more about obstacles, click here.

You will be stopped automatically at the respawn/store point.

If a player reaches the finish position, the game ends.

Character skills

Each characters have 3 own skills.

You can use them after you throw dice.

click here to see details about skills

Targeting skills

If you use skill, avaliable player targets will be shown.

You can attack a target by clicking on it.

You can`t use skill if there is no targets in range.

Non-targeting skills

If you use non-targeting skill, You need to choose which square you will place your skill projectile.

Select the square that your opponent might arrive next turn. After you have chosen a square, click confirm button.

The size of projectile will tell you how many squares will it occupy.

If an opponent arrives on the projectile, the opponent will get the effect.

A projectile successfully placed

For more information about projectile, click here.

The Store

Store button will show up in the right bottom corner if avaliable.

You can choose item with clicking on the list on the left or clicking recommended items.

You can see more about item and stats here.

In this picture, you don`t have enough money to buy the item.
However, you can navigate to the lower level items and buy them first.

If you buy lower level items first, you can upgrade it to upper level item later if you have enough money.

When you upgarde item, total price of lower level items you have will be discounted, and item you bought will replace its lower level items.

To see about item levels, click here.

You can resell your items by clicking the items you have.
The price is 40% of the original price.

End of the game

If one player reaches the end, the game ends.
